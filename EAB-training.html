Content Editor Help

📚 Creating Modules

  • Module Title: Clear, descriptive name (e.g., "Food Safety Basics")
  • Description: Brief overview of what employees will learn
  • Icon: Visual representation to help identify the module
  • Chapters: Break content into digestible learning segments

✏️ Writing Chapter Content

HTML Tags You Can Use:

  • <h3>Section Title</h3> - Section headings
  • <strong>Important Text</strong> - Bold emphasis
  • <ul><li>Item</li></ul> - Bullet lists
  • <ol><li>Step</li></ol> - Numbered lists
  • <p>Paragraph text</p> - Regular paragraphs

Special Boxes:

  • <div class="key-point">Important info</div> - Highlighted key points
  • <div class="tip-box">Helpful tip</div> - Tips and suggestions
  • <div class="warning-box">Warning</div> - Important warnings

❓ Creating Quiz Questions

  • Clear Questions: Ask specific, unambiguous questions
  • Multiple Choice: Provide 2-4 answer options
  • Correct Answer: Select the radio button for the right answer
  • Point Values: Higher points for more important concepts
